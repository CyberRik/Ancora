# ancora (CLI)

Command-line interface for Ancora. **Phase 0 stub** — implements `version`, `info`, and a
`dev` placeholder. The full CLI (`ancora dev` local runtime, run management, logs) lands in
later phases per [`docs/IMPLEMENTATION-PLAN.md`](../../docs/IMPLEMENTATION-PLAN.md).

```bash
ancora version
ancora info
ancora dev      # placeholder until Phase 1
```
